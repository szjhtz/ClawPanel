# ClawPanel Lite v0.1.1

发布时间：2026-03-10

## 模型配置体验优化

- 快速添加模型服务商时不再自动填入一组过时模型 ID
- 新增服务商后默认只创建一个空模型项，用户可以直接填写自己要用的模型 ID
- 减少“先删默认模型再填自己的模型”这类额外操作

## 开箱即用体验继续收口

- 当 OpenClaw 与网关均离线时，面板会明确提示尝试点击左下角手动重启 OpenClaw
- Lite 安装后的主流程继续保持为：安装 → 配置 API Key → 配置通道 → 直接使用
- Lite 版本线更新到 `0.1.1`，文档、安装脚本、更新元数据同步完成

## 发布说明

- Lite 当前版本号：`0.1.1`
- Lite 当前主包：`clawpanel-lite-core-v0.1.1-linux-amd64.tar.gz`
- QQ NapCat 仍维持独立 Bundle 方案，未并入 Lite 主包
