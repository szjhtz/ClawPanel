## v5.2.4 - OpenClaw 桌面端安装前置依赖收口

### 安装体验
- Windows / macOS 安装 OpenClaw 前会先检查 `Node.js >= 20` 与 `Git`
- 缺少依赖时不再尝试自动安装，直接返回清晰错误，避免一键安装半途失败
- 前端阻断提示新增官方链接按钮，用户可直接跳转下载：
  - Node.js: `https://nodejs.org`
  - Git: `https://git-scm.com/downloads`

### 背景
- 之前在桌面端环境里自动安装 Node.js / Git 经常受 winget、PATH、权限、终端环境影响，报错率较高
- 本版本改为“先手动安装基础依赖，再继续一键安装 OpenClaw”，整体成功率和可解释性更好
