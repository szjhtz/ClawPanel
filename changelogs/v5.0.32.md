## v5.0.32 - 修复 NapCat 登录状态误判

### Bug 修复
- **fix**: 修复面板频繁轮询时重复请求 NapCat WebUI 登录，导致 `Unauthorized` 并误判 QQ 为未登录/登录失效的问题
- **fix**: 优化 NapCat 登录状态监控逻辑，遇到一次 `Unauthorized` 时自动 fresh auth 重试，减少登录状态横跳

### 影响范围
- 主要影响 Linux 上通过 Docker 运行 NapCat 的登录状态检测稳定性
- 修复后 `容器在线 + QQ 已登录` 不会再被面板误判成 `login_expired`
