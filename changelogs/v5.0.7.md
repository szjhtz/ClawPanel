# ClawPanel v5.0.7 更新日志

发布日期：2026-02-28

## 修复

- **QQ 插件文件权限**：下载后自动 `chown root:root`，解决 OpenClaw 拒绝加载插件（suspicious ownership）
- **channels.qq 配置错误**：移除 openclaw.json 中手动写入的 `channels.qq`，该频道由 QQ 插件自动注册，手动写入导致 `unknown channel id: qq` 启动错误
- **NapCat WebUI Token 不一致**：修复 Docker 环境变量 `WEBUI_TOKEN` 与 `webui.json` token 不匹配导致扫码登录 Unauthorized
- **napcatAuth() 认证逻辑**：优先从 Docker 容器内 `webui.json` 读取实际 Token，不再仅依赖缓存或环境变量

## 新增

- **NapCat 诊断增强**：新增 Token 一致性检测（Docker 环境变量 vs webui.json）与自动修复
- **config_checker**：新增 `napcat-webui-token-mismatch` 检测项与 `fixNapCatWebUITokenMismatch` 修复函数
- **napcat-fix.sh**：一键修复脚本，适用于 Linux 用户排查 NapCat/OpenClaw 部署问题（11 项检查 + 自动修复）
