# ClawPanel Pro v5.2.8

发布时间：2026-03-11

## 兼容性与探测增强

- 增强 macOS 下 OpenClaw 自动识别，补强 `/Users/*/.openclaw` 与真实用户 home 探测逻辑
- 改善先装 OpenClaw 再装 ClawPanel 时的识别成功率

## 配置与工作区体验修复

- 受管工作区判断兼容 `workspace` / `workspaces` 两种目录命名
- 兼容 `OpenClawWork/workspace/...` 与 `OpenClawWork/workspaces/...` 等结构
- 修复部分 Agent 工作区 / 受管目录场景下编辑器表现异常的问题

## 提供商默认值与安装体验

- 修正 MiniMax 默认 provider 地址为 `https://api.minimaxi.com/anthropic/v1`
- 默认模型改为 `MiniMax-M2.5`
- 面板存在运行中任务时会更及时刷新任务列表与安装进度显示
